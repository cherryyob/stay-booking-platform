# 🏡  - Full-Stack Web App

A functional full-stack replica of the **SEA-StayEnjoyAgain** platform. This project demonstrates server-side logic, relational database management, and modern utility-first styling.

---

## 🌟 Key Features

* **Dynamic Listings:** View properties fetched directly from a **MySQL** database.
* **User Authentication:** Secure login and signup functionality (if implemented).
* **CRUD Operations:** Ability to create, read, update, and delete property listings.
* **Server-Side Rendering:** Dynamic content delivery using **EJS** templates.
* **Tailwind Styling:** A beautiful, responsive UI built with **Tailwind CSS**.
* **Search & Filter:** Find stays based on location or category.

---

## 🛠️ Tech Stack

| Layer | Technology |
| :--- | :--- |
| **Backend** | Node.js, Express.js |
| **Database** | MySQL |
| **Frontend** | EJS (Embedded JavaScript), Tailwind CSS |
| **Tools** | Sequelize/TypeORM (optional), MySQL Workbench |
